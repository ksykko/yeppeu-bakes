### Project Setup
```
npx create-react-app . --use-npm
```

### Install Tailwindcss
```
npm install -D tailwindcss
npx tailwindcss init -p
```

### Install React Router
```
npm install react-router-dom
```

### Install Google Firebase
```
npm install firebase
```
